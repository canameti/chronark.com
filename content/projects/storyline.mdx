## 🚀 Project Overview

As Development Lead, I architected a comprehensive GTA V roleplay server focusing on authentic storytelling and immersive gameplay. The project utilized modern technologies to create a scalable gaming platform with robust features and monitoring.

## 🛠️ Technical Stack

- **Frontend:** Vue.js with TypeScript, custom UI components
- **Serverside:** C#
- **Clientside:** TypeScript/JavaScript
- **Backend:** PHP REST API, MySQL, Redis, JWT
- **Infrastructure:** Docker, Prometheus, Grafana
